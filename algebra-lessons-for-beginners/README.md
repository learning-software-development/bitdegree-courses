## Formulas

1. Area of a Rectangle `A = l * w`
2. Perimeter of a Rectangle `P = 2l + 2w`
3. Distance Traveled (Constant Rate) `D = r * t` rate times time
4. Cost `C = n * p` n is amount of item and p is price per item
